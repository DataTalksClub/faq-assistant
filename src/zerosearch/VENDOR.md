# Vendored: zerosearch

This directory is a **vendored copy** of the standalone `zerosearch` package, so
the dependency-free Cloudflare Python (Pyodide) Worker can bundle it directly
instead of installing it from PyPI at runtime.

- Upstream: https://github.com/alexeygrigorev/zerosearch
- PyPI: https://pypi.org/project/zerosearch/
- Vendored version: **0.1.0** (commit `b818db9`)

Do not edit these files here. To update, bump the upstream package, release it,
then re-copy `zerosearch/` from the upstream repo and update the version above.
